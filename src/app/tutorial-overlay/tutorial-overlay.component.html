Welcome to the App!

The goal of this app is to help you with your mystery shopping! In this short tutorial, we will explain how to use this app.

To navigate through the tutorial, just tap the screen when you are done reading what is on the page.

Here will be a list of all the shops you started

To create a shop use this button

Enter the desired name for the shop here. For this tutorial, we will call it "Demo Shop".

Click this button to create the shop and open it

This is the shop page. This is where you can take notes for the shop you are doing. You can also take a photo, use a timer, or record audio.

In the following pages, we will go over the tools and how they work.

This button is how you add a photo to your notes.

This button is how you start a timer. Once you start a timer, you can add a lap or stop it.

This button is how you add a lap to the timer. The output will show the accumulated time as well as the time between laps.

This button is how you stop the timer, once stopped, you will see the output of the timer, and each lap (if there are any).

This button is how you start an audio recording. Once you start the recording, you can pause it or stop it.

This button is how you pause an audio recording. If the recording is paused, then you have the option to resume it or to stop it.

This button is how you resume an audio recording. Once resumed, you can pause it again or stop it.

This button is how you stop and audio recording. When you stop it, it will output the result. This result will allow you to listen to it over and share it.
